# Figures

This directory contains publication figures for the website.

## Expected files

Place your image files here with the following names:

- `figure1_elife.png` — Figure from your eLife paper (displayed on homepage)
- `figure2.png` — Second figure
- `figure3.png` — Third figure
- `figure4.png` — Fourth figure
- `figure5.png` — Fifth figure

Images can be PNG or JPG format. The website HTML references these filenames.
Update `js/main.js` (`FIGURES` array) if you use different filenames or want to add captions.
